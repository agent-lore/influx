## Codebase Patterns
- LithosClient is an SSE-backed MCP client in `src/influx/lithos_client.py`; exposes `read_note`, `list_notes`, `write_note`, `cache_lookup` etc.
- Contract tests for LithosClient live in `tests/contract/test_lithos_client.py` using a `FakeLithosServer` (FastMCP-based) that records calls and supports response queues
- Config schema uses pydantic v2 models in `src/influx/config.py`; `RepairConfig` already exists with `max_items_per_run = 100`
- Test fixtures in `tests/conftest.py` provide `influx_config_env` (realistic v0.7 TOML) and `fake_lithos_sse_url`
- Use `uv run ruff check`, `uv run ruff format`, `uv run pyright`, `uv run pytest` for quality gates
- Git identity needs explicit config in worktrees: `git config user.email/user.name`
- Repair hook protocols live in `src/influx/repair.py`: `ReExtractArchiveHook`, `Tier2EnrichHook`, `Tier3ExtractHook` — all use `typing.Protocol`
- `ExtractionOutcome` enum (UPGRADE/TERMINAL/TRANSIENT) + `ReExtractionResult` dataclass drive abstract-only re-extraction tag logic
- `sweep(profile, *, client, config)` is the async repair entry point; accepts LithosClient + AppConfig via kwargs
- `list_notes` returns `CallToolResult` whose `.content[0].text` is JSON `{"items": [...]}`; each item has `id`, `title`, etc.
- Unit tests for sweep use `AsyncMock` + `MagicMock` to mock LithosClient (no SSE server needed)
- `select_stages()` accepts extracted `archive_path` and `max_profile_score` as params; caller must parse note content via `parse_note` + `parse_archive_path` + `parse_profile_relevance`
- `StageSelection` frozen dataclass with 5 booleans: `archive_retry`, `text_extraction_retry`, `abstract_only_reextraction`, `tier2_retry`, `tier3_retry`
- `ClearingDecision` frozen dataclass + `compute_clearing()` pure function determine post-stage tag removal for `influx:archive-missing` and `influx:repair-needed`
- Clearing logic: archive-missing iff archive_path stored; repair-needed iff archive_ok AND text_ok AND tier2_ok AND tier3_ok (with terminal exemption waiving Tier 2/3)
